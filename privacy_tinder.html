an1me.to Speed Control – Chrome Extension

Τι δεδομένα συλλέγουμε;

Η επέκταση an1me.to Speed Control δεν συλλέγει και δεν διαμοιράζεται προσωπικά δεδομένα χρήστη. Η μόνη πληροφορία που αποθηκεύεται είναι η προτίμηση ταχύτητας βίντεο, τοπικά στον browser σας μέσω του Chrome Storage API (chrome.storage.local).

Πώς χρησιμοποιούνται τα δεδομένα;

Η προτίμηση ταχύτητας χρησιμοποιείται αποκλειστικά για να προσφέρει την επιλεγμένη εμπειρία αναπαραγωγής στα βίντεο του an1me.to. Τα δεδομένα δεν αποστέλλονται ή κοινοποιούνται σε τρίτους ή οποιοδήποτε διακομιστή.

Πρόσβαση και διαγραφή

Όλες οι ρυθμίσεις διατηρούνται αποκλειστικά στον browser του χρήστη και μπορούν να διαγραφούν ανά πάσα στιγμή με την απεγκατάσταση της επέκτασης ή/και καθαρισμό των δεδομένων του Chrome.
